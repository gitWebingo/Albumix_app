# Albumix - Photo Gallery App

This is a fully functional, offline photo gallery app.

## Features
- **Import Photos**: Select multiple photos from your gallery to import them into Albumix.
- **Organization**: Assign Categories (Albums) and Tags to your photos.
- **View**: Smooth masonry grid layout and zooming capabilities.
- **Storage**: All data is stored locally on the device using Hive and local file storage.

## Architecture
- **State Management**: Provider
- **Local Database**: Hive (NoSQL)
- **UI**: Flutter Material 3 (Dark Theme)

## Getting Started
1. `flutter pub get`
2. `flutter run`
